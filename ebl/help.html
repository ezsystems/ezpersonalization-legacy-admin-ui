Einführung

Die YOOCHOOSE Recommender Engine besteht aus den in der folgenden Zeichnung dargestellten Komponenten, die im Zusammenspiel dafür sorgen, dass aus den Aktivitäten eines Nutzers personalisierte Empfehlungen generiert und ausgespielt werden.

Die Recommender Engine erhält über die Integration des Event-Tracking in die Web-Seiten oder den Online Shop Informationen über die Aktivitäten der Anwender. Dies kann ein Klick auf ein Link sein, der Kauf eines Produktes oder auch eine aktive Bewertung durch den Anwender. Diese Events werden zusammen mit einer User ID (meist eine Cookie-ID) und einem eindeutigen Identifier für den Inhalt der Seite oder das Produkt (z.B. eine Artikelnummer) an die Recommender Engine gesendet (A). Über eine Content-Schnittstelle lernt YOOCHOOSE automatisch, welche Produkte oder Inhalte angeboten werden (B). Bei der Modellberechnung können die Informationen über die Events der User und die vorhandenen Produkte oder Inhalte zur Vorberechnung genutzt werden. Dies geschieht in der Regel jede Nacht automatisch.

Um die berechneten Empfehlungen nutzen zu können, werden diese zur Laufzeit zu einem Ergebnis zusammengebaut (D). Dabei kann zum Beispiel kann ein Fall-Back-Szenario festgelegt werden, das Ergebnisse liefert, falls das im Empfehlungsabruf genutzte Modell kein Ergebnis liefert. Dies kann der Fall sein, wenn ein Artikel oder Produkt noch keinen Event hat und deswegen der Recommendation Engine noch nicht bekannt ist. Der Empfehlungsabruf wird im Normalfall über ein Skript in die eigene Webseite eingebaut und beim Aufruf der Seite durch den Anwender ausgelöst (A). Dabei werden wiederum die User ID und die Item ID übergeben. Das Ergebnis des Empfehlungsabrufs ist eine Liste mit eindeutigen Identifier. Um das Ergebnis für den User darstellen zu können, müssen diese IDs noch über das Content Management System oder den Shop aufgelöst und entsprechende Bilder und Texte angezeigt werden (D).

Wir wünschen viel Erfolg bei Integration der YOOCHOOSE Recommender Engine und freuen uns über Ihr Feedback.
